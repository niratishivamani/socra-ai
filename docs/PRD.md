# Product Requirements Document

## Product Name
Socra AI

## Problem

SOC analysts spend significant time manually investigating alerts across different security tools.

## Goal

Build an AI-powered SOC platform that assists analysts in investigating security alerts.

## Target Users

- SOC Analysts
- Security Analysts
- Cybersecurity Students

## Version 1 Features

- Splunk Integration
- Windows Event Logs
- Sysmon Logs
- MITRE ATT&CK Mapping
- IOC Enrichment
- AI Investigation Chat
- Incident Report Generator

## Out of Scope

- Docker
- Case Management
- Advanced Dashboard
